[![YouTube Video](https://img.youtube.com/vi/rmPn7mWssto/0.jpg)](https://www.youtube.com/watch?v=rmPn7mWssto)

## How to build ##
```
$ cd ~/catkin_ws/src/
$ git clone https://github.com/yzrobot/adaptive_clustering.git
$ cd ~/catkin_ws
$ catkin_make
```
